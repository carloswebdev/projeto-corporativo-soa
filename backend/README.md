## Como rodar

No diretorio raiz, onde contem o pom.xml, utilize o seguinte comando: "mvn jetty:run", isso vai iniciar o jetty em localhost:8080

Para testar se está funcionando, faça uma chamada para a url: 
http://localhost:8080/v1/queroVender/Corsa - Onde "Corsa" é um parametro de busca.

OBS. Será adicionado mais filtros conforme o desenvolvimento ocorre.
